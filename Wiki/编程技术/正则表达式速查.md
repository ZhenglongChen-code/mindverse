# 正则表达式速查

## 核心定义

正则表达式（Regular Expression）是一种用于**匹配字符串模式**的强大工具，广泛用于验证、提取、替换文本。

## 字符类

| 符号 | 含义 | 等价于 |
|------|------|--------|
| `\d` | 任意数字 | `[0-9]` |
| `\w` | 字母数字下划线 | `[A-Za-z0-9_]` |
| `\s` | 空白字符 | 空格/制表符/换行 |
| `.` | 除换行外的任意字符 | - |

## 量词

| 符号 | 含义 |
|------|------|
| `*` | 零个或多个 |
| `+` | 一个或多个 |
| `?` | 零个或一个 |
| `{n}` | 精确n个 |
| `{n,m}` | n到m个 |

## 常用模式

```
邮箱验证：^[\w.-]+@[\w.-]+\.\w+$
手机号：  ^1[3-9]\d{9}$
URL：     https?://[\w.-]+(?:/[\w./-]*)?
```

## 零宽断言（高级）

| 断言 | 名称 | 作用 |
|------|------|------|
| `(?=pattern)` | 正向先行 | 某个位置**后面**符合pattern |
| `(?!pattern)` | 负向先行 | 某个位置**后面**不符合pattern |
| `(?<=pattern)` | 正向后继 | 某个位置**前面**符合pattern |
| `(?<!pattern)` | 负向后继 | 某个位置**前面**不符合pattern |

> 注意：JavaScript 不支持后继断言（`(?<=...)` 和 `(?<!...)`）
